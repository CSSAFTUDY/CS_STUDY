## ✔ Rule

[☉ Issue Template](https://github.com/CSSAFTUDY/CS_STUDY/issues) 이용

```
1. 각자 정해진 주제 조사 후 Issue에 게시
2. Issue에 올라온 정보 모두 숙지
3. 주에 한번, 총 12개의 주제 중 랜덤으로 2개씩 골라 발표
4. 발표를 듣는 팀원들은 주제에 대해 질문
5. 발표 후 알게된 사항들은 Issue 댓글에 정리
```
